What is Swot?

Swot is an application aimed for usage by teachers in Secondary Schools & Sixth Forms. It provides the following features:

  • Management of revision, homework and classwork
  • Calendar integration
  • Intuitive class management through a class code system

How can I use Swot?

If you are looking to make use of Swot in your school you can sign up for a teacher account here.

Who is Swot made by?
